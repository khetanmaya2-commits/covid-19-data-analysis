**# COVID-19 Data Analysis Dashboard**

**## Project Overview**
This project analyzes global COVID-19 data to understand case trends, death rates, and recovery statistics.

#**# Tools Used**
- Python
- Pandas
- Matplotlib
- Google Colab
- Looker Studio

**## Key Analysis**
- Top countries by confirmed cases
- Death rate comparison
- Regional distribution of cases

**## Dashboard**
Interactive dashboard created using Looker Studio.

**## Dataset**
COVID-19 global dataset from Kaggle.
